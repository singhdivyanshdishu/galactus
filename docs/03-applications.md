# Applications Configuration

## Terminal Setup

### Konsole (Primary)
- **Profile**: DankProfile with custom colors
- **Theme**: DankColors.colorscheme
- **Font**: Monospace with good Unicode support
- **Transparency**: Enabled for modern look

### Ghostty (Alternative)
- **Config**: `~/.config/ghostty/config`
- **Theme**: DankColors variant
- **GPU acceleration**: Enabled
- **Modern features**: Ligatures, true color

## File Manager

### Dolphin (Primary)
- **Transparency**: 85% normal, 75% inactive
- **Icons**: Papirus theme
- **Thumbnails**: Enabled for images and videos
- **Integration**: KDE services for advanced features

### Ranger (Terminal)
- **Keybindings**: Vim-style navigation
- **Previews**: Images, text files
- **Integration**: Works with system file associations

## Web Browsers

### Zen Browser (Primary)
- **Source**: Flatpak from Flathub
- **Privacy**: Enhanced tracking protection
- **Performance**: Optimized for speed

### Firefox (Backup)
- **Standard**: Arch repository version
- **Extensions**: Can be customized as needed

## Development Tools

### VS Code
- **Version**: Binary from AUR (visual-studio-code-bin)
- **Extensions**: Install as needed
- **Integration**: Works with system Git

### Terminal Tools
- **Git**: Version control
- **Python**: Latest from repos + pip
- **Node.js**: Latest LTS + npm
- **Build tools**: base-devel package group

## Media Applications

### MPV
- **Configuration**: Minimal, fast video player
- **Formats**: Supports most video/audio formats
- **Integration**: Default video player

### Audio Control
- **Pavucontrol**: GUI audio mixer
- **Playerctl**: Command-line media control
- **Integration**: Works with DMS media keys
