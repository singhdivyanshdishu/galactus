# Application List

## Core Applications

### System
- **niri** - Scrollable tiling Wayland compositor
- **SDDM** - Display manager for login
- **DMS** - Desktop Management System (taskbar, launcher, etc.)

### Terminals
- **Konsole** - Primary KDE terminal with DankProfile theme
- **Ghostty** - Modern GPU-accelerated terminal

### File Management
- **Dolphin** - Primary file manager with transparency effects
- **Thunar** - Lightweight alternative file manager
- **Ranger** - Terminal-based file manager

### Web Browsers
- **Zen Browser** - Privacy-focused browser (Flatpak)
- **Firefox** - Backup browser

### Development
- **Visual Studio Code** - Primary code editor
- **Git** - Version control
- **Python** + pip - Python development
- **Node.js** + npm - JavaScript development

### Media
- **MPV** - Lightweight video player
- **Pavucontrol** - Audio control panel
- **Playerctl** - Media control commands

### Utilities
- **Starship** - Cross-shell prompt
- **Fastfetch** - System information display
- **Cliphist** - Clipboard manager
- **Grim** + **Slurp** - Screenshot tools
- **Htop** - Process monitor
- **Lolcat** - Colorful terminal output

### Themes & Icons
- **Breeze Dark** - System theme
- **Papirus** - Icon theme
- **DankColors** - Custom terminal color scheme

## Package Sources

- **Pacman** - Official Arch repositories
- **AUR** - Arch User Repository (via yay)
- **Flatpak** - Universal packages from Flathub
